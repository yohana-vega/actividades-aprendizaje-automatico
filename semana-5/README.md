# Semana 5

## Aprendizaje Supervisado

En esta semana se trabajó con modelos de aprendizaje automático supervisado utilizando los algoritmos K-NN y Árboles de Decisión.

Se realizó la clasificación de datos utilizando el dataset Wine, aplicando métricas de evaluación, matrices de confusión y visualización de resultados para comparar el rendimiento de ambos modelos.
