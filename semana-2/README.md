# Semana 2 - Adquisición de datos

## Descripción
En esta semana se trabajó con la importación de datos utilizando la librería pandas en Python, a partir de distintos formatos como CSV, JSON y Excel. El objetivo fue comprender cómo cargar datos y realizar operaciones básicas de análisis.

## Actividades realizadas
- Carga de datos desde archivo CSV
- Visualización de las primeras filas del dataset
- Carga de datos desde archivo JSON
- Cálculo de la cantidad total de clientes
- Carga de datos desde archivo Excel
- Cálculo del promedio de stock

## Dataset utilizado
- ventas (archivo .csv)
- clientes (archivo .json)
- inventario (archivo .xlsx)

## Herramientas utilizadas
- Python
- Pandas
- Jupyter Notebook / Google Colab

## Resultados
Se lograron importar correctamente los datos desde diferentes formatos y se obtuvieron resultados básicos como la cantidad de clientes y el promedio de stock.

## Conclusión
La utilización de pandas permite trabajar de manera eficiente con datos de distintas fuentes, facilitando su análisis y preparación para futuros procesos de ciencia de datos.
