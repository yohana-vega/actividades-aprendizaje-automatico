# Semana 2 - Análisis de datos

## Descripción
En esta semana se trabajó con un dataset para realizar un análisis exploratorio de datos (EDA), identificando características principales y posibles problemas en la información.

## Actividades realizadas
- Carga del dataset
- Exploración de datos
- Identificación de variables
- Revisión de valores nulos
- Análisis de distribuciones

## Dataset utilizado
- ventas (archivo .csv)

## Herramientas utilizadas
- Python
- Pandas
- Matplotlib / Seaborn
- Jupyter Notebook / Google Colab
