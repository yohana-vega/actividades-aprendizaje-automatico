## Semana 4 - Aprendizaje Supervisado

### Contenidos

En esta semana se abordaron conceptos de aprendizaje supervisado, enfocados en modelos de regresión y clasificación.

Se trabajó con:
- Regresión lineal (variable continua)
- Regresión logística (variable discreta)
- Entrenamiento de modelos con scikit-learn
- Predicción de resultados
- Evaluación de modelos mediante métricas de desempeño

---

### Actividades realizadas

**Actividad 1 - Regresión Lineal**
Se aplicó un modelo de regresión lineal utilizando el dataset Boston Housing para predecir el precio de viviendas.

**Actividad 2 - Regresión Logística**
Se aplicó un modelo de regresión logística para clasificar el sistema operativo de usuarios a partir de su comportamiento en un sitio web.

---

### Herramientas utilizadas

- Pandas  
- Matplotlib  
- Scikit-learn
