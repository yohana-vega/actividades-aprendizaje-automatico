# Semana 4 - Aprendizaje Supervisado

## Descripción
En esta semana se trabajó con modelos de aprendizaje supervisado, aplicando regresión y clasificación para realizar predicciones a partir de datos.

## Actividades realizadas
• Preparación de datos
• Creación de conjuntos de entrenamiento y prueba
• Aplicación de regresión lineal
• Aplicación de regresión logística
• Evaluación de modelos

## Herramientas utilizadas
• Pandas
• Matplotlib
• Scikit-learn
